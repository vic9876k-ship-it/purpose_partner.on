## Getting Started

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Register a new account or login
4. Start setting goals and finding accountability partners!

## PWA Installation

This app can be installed as a PWA on supported devices:
- Chrome/Edge: Look for "Install" in the address bar
- Mobile: Add to home screen from browser menu
